#  :currency_exchange: Conversor de Moedas

<p>Aplicação web que realiza a conversão de Real (BRL) para Dólar (USD), Euro (EUR) e Bitcoin (BTC) utilizando dados atualizados em tempo real por meio do consumo de API.</p>
<p> :round_pushpin: O projeto foi construído com foco em:</p>
<ul>
  <li>Integração com API externa</li>
  <li>Manipulação dinâmica do DOM</li>
  <li>Organização de código</li>
  <li>Experiência do usuário</li>
  <li>Responsividade e adaptação multiplataforma</li>
</ul>

🔗 Deploy: https://marianaasoares.github.io/conversor-de-moedas/

📁 Repositório: https://github.com/MarianaASoares/conversor-de-moedas
